This component provides the labels helper function to other components via the context API. Currently, defined labels can
be found [here](/docs/tedi-ready-providers-labelprovider-labels--labels)

## Usage

### Provider

Wrap the provider around your app.

```jsx
<LabelProvider labels={labels} locale={locale}>
  <YourApp />
</LabelProvider>
```

`Label` is an object of key-value pairs. Keys should follow this pattern:

<ul>
  <li>`<labelName>`</li>
  <li>`<componentName>.<labelName>`</li>
</ul>
Values can be either plain strings or, in some cases, functions. Functions can be used for conditional translations, such as plurals.

### Consumer

You can use the hook:

```js
const { getLabel } = useLabels();

// String label accepts only key
const label = getLabel('someKey');

// Function label accepts key and arguments
const label = getLabel('someKey', 'Username')
```

Or wrap your component in the context consumer:

```jsx
<LabelContext.Consumer>{(labels) => <Component someLabel={labels.getLabel('someKey')} />}</LabelContext.Consumer>
```
